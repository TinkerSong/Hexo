# 算术运算符:
```
加法 +
减法 -
乘法 *
除法 /
取余 %
```
```
⾃增1 ++
 a++ 就是 a=a+1
⾃减1 --
 a-- 就是 a=a-1
int a = 5;
int b = a++;
int c = ++a;
int d = a--;
int e = --a;
```
⾃增（++）⾃减（--）运算符是⼀种特殊的算术运算符
⾃增（++）⾃减（—）在变量左右有不同的意义
在变量左边则先⾃增减操作，再进⾏表达式运算
在变量右边则先进⾏表达式运算，在进⾏⾃增减操作
⾃增和⾃减运算符只能⽤于操作变量，不能直接⽤于操作数值或常量
