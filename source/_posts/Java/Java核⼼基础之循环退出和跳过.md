# 循环退出
break 关键字
跳出当前层的循环体，执⾏循环体下⾯的语句
重要：字符串⽐较是否相等，不能⽤ ==，应该是⽤ equals⽅法
# 循环跳过
continue 关键字
跳过当前循环，执⾏下⼀次循环, (忽略当前循环内容，直接执⾏下⼀次)
